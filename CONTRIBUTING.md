# Contributing Guidelines

Thanks for your interest in this project!

This repository is mainly used for:
- Course study and experiment reproduction
- Materials data mining practice
- Machine learning model exploration

Although this is a course project, contributions are welcome.

---

## How to Contribute

You can contribute in the following ways:

- Reporting bugs or issues
- Suggesting improvements to code structure or documentation
- Improving experiment reproducibility
- Optimizing feature engineering or model evaluation methods

---

## Before Submitting

Before submitting a pull request, please make sure that:

1. The code runs without errors
2. Jupyter notebooks can be executed from top to bottom
3. Relative paths are used for datasets
4. New dependencies are clearly documented

---

## Communication

- Please open an Issue for discussion before submitting major changes
- Be respectful and constructive in all discussions

Thank you for your contribution!
