# Course Project

本目录为《材料数据挖掘利用方法》课程的**课程大作业（Course Project）**，是本仓库的核心部分。

## 主要内容

- 数据预处理与特征工程
- 特征选择方法（互信息、随机森林特征重要性等）
- 回归模型构建（随机森林回归、岭回归等）
- 模型评估与结果分析
- 模型解释性分析（如 SHAP）

## 目录结构说明（示例）

- `code/`：实验与建模代码
- `report/`：课程大作业报告（PDF）
- `slides/`：汇报用 PPT

📌 **说明**：
本目录中的内容展示了完整的材料数据挖掘建模流程，
建议作为理解本课程核心方法的主要入口。
