# Datasets

本目录用于存放课程实验与课程大作业中使用的**数据集及相关说明文件**。

## 数据类型说明

- 原始数据集（如 `.xlsx` / `.csv`）
- 用于可视化或分析的派生数据
- 数据说明文档（如 PDF）

📌 **注意事项**：

- 若数据来源受限或不便公开，可仅保留数据格式说明
- 在运行代码前，请确认数据文件路径与代码中一致
- 建议不要在仓库中提交体积过大的原始数据

📌 **命名建议**：
- `project_target_dataset.xlsx`
- `dataset_for_visualization.xlsx`
- `*_description.pdf`
